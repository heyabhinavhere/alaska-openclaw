## Amplitude API breakthrough
- **The filter operator is `"greater"` NOT `"greater than"`.** The API returns HTTP 400 for "greater than". This is why all DAU numbers were inflated 3-5x.
- Same for `"less"` (not "less than"), `"greater or equal"`, `"less or equal"`.
- `curl --data-urlencode` also fails with complex nested JSON. Must use python3 urllib for filtered queries.
- Verified Real Users filter works: Apr 21 = 12 DAU (matches Abhinav's dashboard), vs 55 unfiltered.
- Updated: amplitude-api-reference.md, amplitude-analyst SKILL.md v2.0, Daily Pulse cron prompt.

## Lesson learned
- **Never silently degrade.** If a critical dependency is missing, FAIL LOUDLY (alert humans) instead of producing stale output.
- **Git commit critical files immediately.** Gateway restarts wipe uncommitted workspace files.
- **Test the full chain.** We updated all the cron prompts but never verified the actual file existed on disk.
